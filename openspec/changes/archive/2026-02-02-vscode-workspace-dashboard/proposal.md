## Why

There are many VS Code workspaces throughout my system and I often lose track where they are

## What Changes

A Dashboard that can show in a user-friendly way the data saved in the VS Code's Workspace State documents

## Capabilities

### New Capabilities

- `basic-view-dashboard`: Loads all json files in the workspace state folder and displays them in a user-friendly way

### Modified Capabilities

## Impact

Accesses local filesystem (read only)
Runs server (only accessible to localhost)
